# Data Fixtures

Базовые наборы данных, используемые для интеграционных тестов и демонстраций.

## Правила
- Каждая фикстура должна ссылаться на конкретную версию шаблона игры или API.
- Используйте человекочитаемые файлы (JSON/YAML) с комментариями в сопроводительном README.
- При обновлении фиксируйте изменения в changelog шаблонов или сервисов.
- Не храните PII; используйте вымышленные имена и данные.
- Добавляйте владельца и дату последнего обновления в заголовке файла.
