# Mocks and Stubs

Моки и заглушки для движка, Router и внешних сервисов до подключения реальных интеграций.

## Правила
- Все моки должны иметь связку с записью в `docs/legacy/debt-log.csv` и шаблоном удаления заглушки.
- Храните моки в подпапках по компонентам (`llm/`, `router/`, `editor/` и т.д.).
- Для сложных ответов добавляйте README с описанием сценария и шагов воспроизведения.
- Не коммитьте реальные токены/секреты.
- Перед релизом сверяйте, что моки не попадают в prod-сборки.
