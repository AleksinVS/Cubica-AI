# Телеметрия Game Editor (черновик)

## Назначение
Телеметрия (telemetry — технические события и метрики работы приложения) нужна для:
- понимания поведения авторов (какие сценарии используются);
- диагностики ошибок (без утечки секретов/контента);
- оценки производительности UI.

## MVP-правила
- Не отправлять секреты и ключи доступа.
- Для контента использовать хэши/идентификаторы, а не полный текст.
- Все события должны иметь `editor_session_id`, `user_id` (если есть), `timestamp`.
