# Промпты Game Editor (черновик)

Промпт (prompt — текст/структура запроса к LLM) должен быть:
- воспроизводимым (можно повторить сценарий);
- безопасным (не включать секреты);
- версионированным (изменения фиксируются).

На этом этапе документ задаёт правила, а сами промпты будут вынесены в отдельный каталог при реализации Editor/Engine.
