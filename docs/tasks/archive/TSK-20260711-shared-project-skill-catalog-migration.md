# TSK-20260711-shared-project-skill-catalog-migration: Единый каталог навыков

## Status

done

## Parent

none

## Why

ADR-070 устанавливает единый канонический каталог skills/ для проектных навыков. Нужна миграция активных навыков, чтобы все агенты читали один и тот же текст, а Codex сохранял автоматическое обнаружение через символические ссылки.

## Scope

Входит в работу:

- перенести все активные проектные навыки в skills/;
- присвоить внутренним навыкам канонические имена C_*;
- создать мосты обнаружения Codex без копирования файлов;
- добавить общий реестр, JSON Schema и проверку топологии;
- обновить пути в адаптере внешних навыков, документации и проверках.

Не входит в работу:

- изменение содержания предметных навыков;
- адаптация новых внешних навыков;
- изменение правил включения $cubica.

## Acceptance

- Каждый активный проектный навык имеет ровно одну физическую каноническую копию в skills/.
- Внутренние навыки имеют каноническое имя C_*; адаптированные сохраняют предметное имя и происхождение.
- .codex/skills/ содержит только символические мосты и не содержит самостоятельных навыков.
- Реестр и JSON Schema описывают все активные навыки.
- Внешний реестр ссылается на канонические пути, а его валидатор проходит.
- $cubica остается явно включаемым, остальные допущенные навыки сохраняют прежний режим включения.
