# FEATURE: Game Editor — Preparatory Dialog

## Контекст и цели

Подготовительный этап создания игры — диалог пользователя с ИИ-агентом в боковой панели чата. На этом этапе пользователь определяет ключевые параметры будущей игры через структурированный диалог.

- [ ] Определить бизнес-цели игры (на что нужно повлиять в работе)
- [ ] Определить учебные цели игры (какие выводы должен сделать игрок, какие инструменты отработать)
- [ ] Подобрать игровые действия (механики) для достижения учебных целей
- [ ] Выбрать жанр игры и написать сюжет игры

## Объём

**In scope:**
- Боковая панель чата для диалога с ИИ-агентом
- Skills для ИИ-агента (определение целей, подбор механик, выбор жанра)
- Промты для направления пользователя через этапы подготовки
- Сохранение результатов диалога в структурированном формате

**Out of scope:**
- Визуальный редактор (это следующие этапы)
- Автоматическая генерация полного сценария (только подготовка параметров)

## Задачи

- [ ] Разработать UI боковой панели чата
  - [ ] Компонент чата с историей сообщений
  - [ ] Поле ввода сообщений
  - [ ] Индикатор состояния агента

- [ ] Подготовить skills для ИИ-агента
  - [ ] Skill: определение бизнес-целей
  - [ ] Skill: определение учебных целей
  - [ ] Skill: подбор игровых механик
  - [ ] Skill: выбор жанра и формирование сюжета

- [ ] Создать промты для направления диалога
  - [ ] Промт для начала диалога (приветствие, объяснение процесса)
  - [ ] Промты-переходы между этапами
  - [ ] Промт для подведения итогов этапа

- [ ] Реализовать сохранение результатов
  - [ ] Структура данных для хранения результатов подготовительного этапа
  - [ ] Интеграция с манифестом игры

## Acceptance Criteria

- [ ] Пользователь может вести диалог с ИИ-агентом в боковой панели
- [ ] Агент последовательно проводит пользователя через все этапы подготовки
- [ ] Результаты каждого этапа сохраняются и доступны для следующих этапов
- [ ] Пользователь может вернуться и изменить ранее определённые параметры
