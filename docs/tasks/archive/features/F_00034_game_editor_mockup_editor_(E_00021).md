# FEATURE: Game Editor — Mockup Editor

## Контекст и цели

Этап создания интерфейса игры. В поле редактора отображается мокап (макет) экрана игры, размеченный в JSON. Пользователь может:
- Выделить элемент мокапа щелчком мыши
- Выделить область мокапа растягиванием прямоугольника
- Попросить LLM изменить выделенный элемент или область

При выделении элемента/области рядом с курсором мыши появляется активное поле ввода текста, в которое можно сразу начать печатать запрос к LLM.

- [ ] Реализовать отображение мокапа из JSON-разметки
- [ ] Добавить выделение элементов и областей
- [ ] Реализовать inline-поле ввода для запросов к LLM

## Объём

**In scope:**
- Рендеринг мокапа из JSON-разметки
- Выделение элементов мокапа кликом
- Выделение произвольной области прямоугольником
- Inline-поле ввода для запросов к LLM (появляется при выделении)
- Интеграция с LLM для изменения элементов
- Сохранение изменений в JSON-разметку

**Out of scope:**
- Drag-and-drop перемещение элементов
- Создание элементов с нуля (только редактирование существующих)
- Сложные стили и анимации
