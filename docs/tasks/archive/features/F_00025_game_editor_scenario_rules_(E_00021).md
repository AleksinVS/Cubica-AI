# FEATURE: Game Editor — Scenario & Rules Editor

## Контекст и цели

Этап работы над сценарием и правилами игры. Пользователь продолжает общаться с ИИ-агентом в чате, при этом в поле редактора отображается текстовый редактор создаваемых файлов. Агент создаёт файлы сценария и правил, пользователь может их редактировать.

Ключевая функция — отображение сценария в виде flow-диаграммы (диаграммы потока) в окне редактора. Для этого необходимо продумать структуру хранения данных диаграммы в манифесте игры.

- [ ] Реализовать текстовый редактор файлов сценария и правил
- [ ] Создать визуализацию сценария как flow-диаграммы
- [ ] Определить структуру данных flow-диаграммы в манифесте

## Объём

**In scope:**
- Текстовый редактор для файлов сценария и правил
- Визуализация сценария как flow-диаграмма (узлы, переходы, условия)
- Структура данных flow-диаграммы в манифесте игры
- Синхронизация между текстовым представлением и диаграммой
- Взаимодействие с ИИ-агентом для создания/редактирования сценария

**Out of scope:**
- Визуальное редактирование диаграммы drag-and-drop (опционально для MVP)
- Сложные условия ветвления (только базовые переходы)

## Задачи

- [ ] Разработать текстовый редактор
  - [ ] Компонент редактора с подсветкой синтаксиса
  - [ ] Поддержка множественных файлов (вкладки)
  - [ ] Сохранение и загрузка файлов

- [ ] Разработать визуализацию flow-диаграммы
  - [ ] Выбор библиотеки для рендеринга диаграмм (например, React Flow, D3)
  - [ ] Компонент отображения диаграммы
  - [ ] Рендеринг узлов (сцены, события, диалоги)
  - [ ] Рендеринг переходов (стрелки, условия)

- [ ] Определить структуру данных в манифесте
  - [ ] Схема узлов (id, type, content, position)
  - [ ] Схема переходов (source, target, condition)
  - [ ] Схема метаданных диаграммы (viewport, zoom, layout)
  - [ ] Документировать формат в ADR

- [ ] Реализовать синхронизацию
  - [ ] Парсинг текстового сценария в структуру диаграммы
  - [ ] Генерация текста из структуры диаграммы
  - [ ] Двусторонняя синхронизация при изменениях

- [ ] Интегрировать с ИИ-агентом
  - [ ] Агент создаёт файлы сценария
  - [ ] Агент предлагает структуру flow-диаграммы
  - [ ] Пользователь может запросить изменения через чат

## Acceptance Criteria

- [ ] Пользователь может редактировать текст сценария в редакторе
- [ ] Сценарий автоматически визуализируется как flow-диаграмма
- [ ] Структура диаграммы сохраняется в манифесте игры
- [ ] ИИ-агент может создавать и модифицировать сценарий по запросу
