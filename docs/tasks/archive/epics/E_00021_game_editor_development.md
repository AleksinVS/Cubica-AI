# EPIC: Game Editor Development (MVP)

## Контекст

Создание MVP графического редактора игр (Game Editor) — ключевой инструмент для разработки обучающих игр на платформе Cubica. Game Editor представляет собой интерфейс с боковой панелью чата (для взаимодействия с ИИ-агентом) и полем редактора (для визуальной работы с контентом игры).

Редактор должен поддерживать полный цикл создания игры:
1. Определение бизнес-целей и учебных целей
2. Проработка сценария и правил игры
3. Создание интерфейса (мокапов)
4. Сборка прототипа
5. Отладка игры

## Цели и метрики

- [ ] Пользователь может создать полный сценарий игры с помощью диалога с ИИ-агентом
- [ ] Сценарий отображается как flow-диаграмма в редакторе
- [ ] Мокап экрана можно редактировать через выделение элементов и текстовые команды LLM
- [ ] Сверстанный прототип отображается в редакторе с возможностью итеративных изменений
- [ ] Игра запускается в режиме отладки внутри редактора

## Объём

**In scope:**
- Архитектура и базовая структура Game Editor
- Диалоговый интерфейс с ИИ-агентом (боковая панель чата)
- Текстовый редактор файлов сценария и правил
- Flow-диаграмма сценария в редакторе
- Редактор мокапов с выделением элементов и областей
- Прототип с версткой и итеративными изменениями
- Режим отладки (запуск игры в редакторе)
- Skills и промты для ИИ-агента

**Out of scope:**
- Полноценная IDE с дебаггером кода
- Мультипользовательское редактирование (collaborative editing)
- Интеграция с внешними системами контроля версий
