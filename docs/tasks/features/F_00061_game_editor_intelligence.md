# F_00061: Game Editor Intelligence

- **Эпик**: [E_00021](../epics/E_00021-game-editor.md)
- **Статус**: Planned
- **Приоритет**: High

## Описание
Расширение возможностей Game Editor с использованием продвинутых LLM-функций для помощи разработчику.

## Проблема
Редактирование JSON-манифестов вручную сложно и подвержено ошибкам. Текущий "LLM-first" подход требует умных инструментов для генерации валидного контента.

## Задачи
1.  Реализовать **Schema-aware prompting**: LLM должна знать актуальную схему манифеста при генерации.
2.  Внедрить валидацию сгенерированного контента "на лету" с обратной связью для LLM (Self-correction).
3.  Реализовать генерацию Unit-тестов для скриптов на основе их описания.

## Acceptance Criteria
- [ ] Редактор автоматически исправляет структурные ошибки в JSON, предлагаемые LLM.
- [ ] Редактор умеет генерировать валидные скрипты JS на основе текстового описания.
