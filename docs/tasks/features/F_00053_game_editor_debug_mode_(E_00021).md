# FEATURE: Game Editor — Debug Mode

## Контекст и цели

Этап отладки игры. Игра запускается в окне редактора для тестирования и отладки. Окно чата становится сворачиваемым, чтобы дать больше места для игры. Пользователь может играть в игру и при необходимости обращаться к ИИ-агенту для помощи.

- [ ] Реализовать запуск игры в окне редактора
- [ ] Добавить сворачиваемое окно чата
- [ ] Добавить базовые инструменты отладки

## Объём

**In scope:**
- Запуск игры в iframe редактора
- Сворачиваемая панель чата (collapse/expand)
- Базовые инструменты отладки:
  - Консоль логов
  - Просмотр состояния игры (state inspector)
  - Перезапуск игры
  - Переход к определённой сцене

**Out of scope:**
- Полноценный дебаггер с breakpoints
- Профилирование производительности
- Тестирование мультиплеера

## Задачи

- [ ] Реализовать запуск игры
  - [ ] Сборка игры для запуска (bundling)
  - [ ] Компонент iframe с игрой
  - [ ] Коммуникация между редактором и игрой (postMessage)

- [ ] Реализовать сворачиваемый чат
  - [ ] Кнопка сворачивания/разворачивания панели чата
  - [ ] Анимация сворачивания
  - [ ] Сохранение состояния (развёрнут/свёрнут)

- [ ] Реализовать инструменты отладки
  - [ ] Панель консоли логов
    - [ ] Перехват console.log из игры
    - [ ] Отображение логов с фильтрацией (info, warn, error)
  - [ ] State inspector
    - [ ] Отображение текущего состояния игры
    - [ ] Древовидная структура данных
    - [ ] Редактирование значений (hot reload)
  - [ ] Управление игрой
    - [ ] Кнопка перезапуска
    - [ ] Выпадающий список сцен для перехода
    - [ ] Пауза/продолжение (если применимо)

- [ ] Интегрировать с ИИ-агентом
  - [ ] Агент может видеть логи и состояние игры
  - [ ] Пользователь может спросить агента о проблемах
  - [ ] Агент может предложить исправления
