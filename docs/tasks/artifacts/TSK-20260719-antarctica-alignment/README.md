# Артефакты TSK-20260719-antarctica-alignment

Постоянные артефакты задачи выравнивания игры «Антарктика» по UI-эталону,
целевой архитектуре и эталонной игровой логике.

Состав (файлы появляются по мере выполнения блоков задачи):

- `ui-gap-register.md` — измеренный реестр расхождений web-интерфейса с
  эталоном `draft/antarctica-nextjs-player_AI`; каждая запись классифицирована
  как дефект, шум рендеринга или намеренное отличие.
- `reference-logic-spec.md` / `reference-logic-spec.json` — эталонная
  спецификация игровой логики, извлечённая из монолита `draft/Antarctica`
  (экраны, карточки, метрики, переходы, случайность, окончания) и её
  машиночитаемая проекция.
- `current-logic-projection.md` / `current-logic-projection.json` — проекция
  текущей исполнимой логики из `games/antarctica/game.manifest.json` в той же
  схеме, что эталонная (для честного диффа).
- `logic-gap-register.md` — реестр расхождений текущей логики
  (`games/antarctica/game.manifest.json`) с эталонной спецификацией.
- `architecture-gap-register.md` — реестр расхождений контура Antarctica с
  целевой архитектурой платформы.

Временные материалы сравнения (снимки, диффы, профили) живут в
`.tmp/ui-compare/` и не коммитятся.
