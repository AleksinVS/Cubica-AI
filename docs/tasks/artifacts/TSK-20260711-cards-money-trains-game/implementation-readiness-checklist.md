# Контрольный список готовности реализации «Карты, деньги, поезда»

Список используется оркестратором и исполнителями как проверка входов и результатов корневой задачи. Он не заменяет план TSK и не разрешает самостоятельно принимать архитектурные решения.

## 1. До изменения контрактов

- [x] PM подтвердил минимальное решение ведущего; ADR-071 принят.
- [x] PM подтвердил динамическое строительство road/waypoint; ADR-072 принят.
- [x] Статусы принятых ADR синхронизированы с `PROJECT_ARCHITECTURE.md`.
- [x] PM подтвердил ADR-074 о безопасных ссылках на созданные в сессии объекты.
- [ ] Для изменения выбрана только возможность ближайшего GSR.
- [ ] Классификация «общая / только для игры» записана в задаче исполнителя.
- [ ] Контракт сначала спроектирован в JSON Schema; императивный валидатор не является источником истины.
- [ ] Назначен один владелец общего схемного изменения, чтобы параллельные исполнители не создавали разные формы.

## 2. Для каждого платформенного среза

- [ ] Есть нейтральная фикстура без названий, цветов и терминалов этой игры.
- [ ] Есть позитивная и негативная схема/контрактная фикстура.
- [ ] Есть runtime-тест успешного действия и атомарного отказа.
- [ ] Presenter не раскрывает секреты.
- [ ] Доступное действие проецируется декларативно.
- [ ] В generic runtime/player нет условия по идентификатору игры.
- [ ] Документация описывает фактическую форму после реализации.
- [ ] Generated contracts/manifests пересобраны и проверены на рассинхронизацию.

## 3. Перед GSR-030

- [ ] Передана стартовая сеть демонстрации.
- [ ] Однозначно указаны связь и координаты `3 ↔ π`.
- [ ] Переданы выплаты демонстрационных грузов.
- [ ] Передана ожидаемая последовательность действий локомотивов.
- [x] Определено поведение при недостатке денег: действие отклоняется, кредит/минус запрещены, продажа возможна только в рынке.
- [ ] Карта зарегистрирована с происхождением и правами.
- [ ] Готовы команды/фазы/деньги/граф/роль ведущего/Phaser-host.

## 4. Перед GSR-031

- [ ] Передан контрольный набор новостей с длительностью и эффектами.
- [ ] Передан контрольный набор грузов и количество карт.
- [ ] Передана обычная конфигурация команд; базовый старт уже закреплен: 10 монет + 2 вагона логистике / 1 локомотив гильдии.
- [ ] Колоды используют воспроизводимую случайность и скрывают будущий порядок.
- [ ] Рынок, выбор груза, движение и расчеты имеют контрольный транскрипт.

## 5. Перед GSR-032

- [ ] Переданы канонические координаты начальных узлов и проверяемые контуры областей.
- [ ] Переданы ограничения строительства и правила временного закрытия; отсутствие дополнительных ограничений указано явно.
- [ ] Контрольный сценарий подтверждает атомарное списание только полной распределенной стоимости.
- [ ] Определена точная граница открытия дороги.
- [ ] Есть груз, для которого новая сеть меняет маршрут или доступность.
- [ ] Контрольная фаза строит минимум полустанок и дорогу до отдельного действия завершения этапа.

## 6. Перед GSR-033

- [ ] Передан полный контент одного континента.
- [ ] Переданы допустимые конфигурации группы и правила равенства.
- [ ] Переданы инструкции ведущего/участника и методические приложения.
- [ ] Сессия хранится постоянно и восстанавливается после перезапуска.
- [ ] Версия игрового пакета закреплена за сохраненной сессией.
- [ ] Фикстуры начала, середины и конца полной партии готовы.

## 7. Перед заявлением о завершении

- [ ] GSR-030–033 имеют фактические доказательства приемки.
- [ ] Полный сокращенный replay — повторное воспроизведение партии по транскрипту — проходит детерминированно.
- [ ] Browser-проверки покрывают мышь и действия без drag/drop.
- [ ] Повторное монтирование Phaser не оставляет обработчики или ресурсы.
- [ ] Перезапуск runtime не меняет состояние и будущий порядок колод.
- [ ] Финальный расчет объясним и определяет двух победителей или равенство.
- [ ] Нет второго континента, штрафа за отчетность и скрытой персональной аналитики.
- [ ] Все временные ограничения зарегистрированы или сняты.
- [ ] `PROJECT_ARCHITECTURE.md`, GSR, TSK, матрица требований и структура репозитория синхронизированы.
