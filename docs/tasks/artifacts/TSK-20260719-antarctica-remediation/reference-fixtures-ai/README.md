# Сохранённый эталонный контент приложения `antarctica-nextjs-player_AI`

## Происхождение

Этот каталог содержит архивированный контент из приложения `draft/antarctica-nextjs-player_AI`, которое было удалено 2026-07-19 в соответствии с решением R1 задачи `TSK-20260719-antarctica-remediation`.

Полная история приложения сохранена в git до момента удаления (см. commit 04a2be6 и позже).

## Назначение

Сохранённый эталонный контент журнала (journal) и левой боковой панели (leftsidebar), с которым сверялась реализация при разработке `apps/player-web`.

## Содержимое

- **screen_j.json** — фикстура журнала ходов (257 строк, заполнена), содержит 2 записи журнала и 16 journalVariableComponent.
- **screen_leftsidebar.json** — фикстура leftsidebar-раскладки (575 строк, заполнена).
- **journal-ref.png** — скриншот эталонного отрисовки журнала (256KB).
- **leftsidebar-ref.png** — скриншот эталонного отрисовки leftsidebar (301KB).

## Отличия от `game-player-nextjs`

Более детальное описание см. в `R1-references-facts.md`:

- **Журнал**: тексты записей и значения метрик отличаются (см. `diff screen_j.json`). Измеренный pixel mismatch: 2,36% (различия обусловлены разным содержимым фикстур, не структурой/стилями).
- **Leftsidebar**: 3 расхождения в ключе (`"command"` в `_AI` vs `"proc"` в `game-player-nextjs`), обусловленные разными версиями схемы экшенов. Pixel mismatch: 0,00% (идентичны до пикселя).

## S1 и hint

Это приложение содержало только пустые заглушки фикстур S1 (`screen_s1.json`) и hint (`screen_hint.json`). Заполненные версии этих экранов находятся в `draft/game-player-nextjs` и сохранены там как эталон.
