# Cubica SDK Core

Базовый слой для всех клиентских SDK Cubica: общие контракты, управление сессиями, сетевой транспорт и правила ретраев.

## Структура
- `src/` — TypeScript-реализация транспортов и контрактов.
- `tests/` — тесты для транспорта, сериализации и управления сессиями.
- `docs/` — дополнительные заметки (схемы, ADR ссылки).

## Как использовать
1. Установить зависимости (например, `pnpm install`).
2. Разрабатывать новые API в `src/`, синхронизируя контракты с Router/Engine.
3. Добавлять тесты в `tests/`, покрывая happy-path и fallback сценарии.
4. Перед релизом обновить документацию и версии зависимых пакетов (React SDK, специализированные SDK).
