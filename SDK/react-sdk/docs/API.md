# React SDK API Overview

## Хуки
- `useCubicaSession(options)` — подключение к сессии, возврат статуса и действий.
- `useCubicaAction()` — отправка действий от имени игрока (определить сигнатуру при реализации).

## Компоненты
- `<CubicaProvider>` — оборачивает приложение, управляет зависимостями core/shared.
- `<GameCanvas />` — базовый визуальный компонент (см. `src/ui/GameCanvas.tsx`).

_Документ обновляется по мере реализации. Все изменения должны ссылаться на версии Router/Engine._
