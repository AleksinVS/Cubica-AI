# React SDK Examples

Планируемые примеры:
- `basic-next-app` — простое подключение через Next.js.
- `storybook-scenarios` — демонстрация UI компонентов из `@cubica/sdk-shared`.
- `custom-actions` — пример расширения хуков дополнительной логикой.

Каждый пример должен описывать зависимости, команды запуска и ожидаемые результаты.
