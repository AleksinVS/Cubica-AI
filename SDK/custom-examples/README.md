# Custom SDK Examples

Каталог для специализированных SDK-интеграций (например, корпоративные клиенты, внутренние демо).

## Как структурировать
- Создавайте подпапку вида `example-name/` с собственным `package.json` и документацией.
- Подключайте `@cubica/sdk-core` и `@cubica/sdk-shared` через workspace либо локальные ссылки.
- Добавляйте README с описанием сценария, зависимостей и инструкций по запуску.

## Требования
- Каждая реализация должна ссылаться на релевантные ADR и записи в `docs/legacy/debt-log.csv`, если используются заглушки.
- Тесты/демо должны быть воспроизводимыми (указать команды `pnpm install`, `pnpm dev`).
- При публикации в репозиторий описывайте лицензионные ограничения и владельцев контента.
