# SDK simulators

Каталог симуляторов и тестовых клиентов, которые помогают проверять интеграцию SDK с Router/Engine без полного фронтенда.

## Практика
- Подпапка `simulator-name/` с простым UI или CLI.
- Используйте core/shared пакеты для повторного использования контрактов.
- Документируйте предусловия (моки, docker-compose) и команды запуска.
