# Cubica SDK Shared

Набор UI-компонентов, тем и утилит, которые переиспользуются клиентскими SDK (React, Telegram, Phaser и др.).

## Структура
- `src/` — компоненты, хелперы и темы.
- Компоненты: `GameScreen`, `GameArea`, `GameButton`, `GameCard`, `GameVariable`, `JournalVariable`, `HelperComponent`. Они принимают `actions` (`command/payload`) и передают события презентеру через `onAction`.
- `tests/` — визуальные/юнит тесты и снапшоты.
- `docs/` — инструкции по дизайну и интеграции.

## Рабочий процесс
1. Создавайте компоненты в `src/`, придерживаясь дизайн-токенов платформы.
2. Поддерживайте сторибуки/демо (добавить позже) и описывайте контракт компонента в документации.
3. При изменении API уведомляйте команды, использующие React SDK и специализированные клиенты.
4. Поддерживайте соответствие требованиям доступности и i18n.
5. Для анимации метрик (`GameVariable`) используется `framer-motion` как peer dependency: подключайте его в потребителе пакета.
