## Назначение

Каталог содержит локальные debug-скрипты (вспомогательные скрипты для ручной диагностики), перенесенные из корня репозитория в рамках `TSK-20260518-architecture-repair-and-task-system-migration`.

Скрипты помогают проверять визуальные состояния `apps/player-web`, журнал ходов и отдельные runtime-сценарии. Они не входят в canonical checks; для обязательной проверки используйте `npm run verify:canonical`.

## Правила использования

- Запускайте скрипты вручную только для расследования конкретной регрессии.
- Для прямых runtime-вызовов используйте `runtime-command-client.cjs`: он
  сохраняет одноразовый Bearer-токен только в процессе и последовательно
  обновляет версию состояния. Проверки через player-web выполняют BFF-запросы
  внутри страницы Playwright, чтобы защищённая cookie оставалась в браузере.
- Временные результаты сохраняйте в `.tmp/`.
- Если скрипт становится обязательной проверкой, добавьте отдельный npm script и опишите его в документации.
