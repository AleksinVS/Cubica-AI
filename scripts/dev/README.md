# Скрипты разработки

Каталог содержит скрипты для локальной разработки, запуска окружения и вспомогательных операций.

## Рекомендации
- Не храните секреты в скриптах; используйте переменные окружения.
- Перед использованием в команде проверяйте скрипты на чистой машине или dev-контейнере.
- Для сложных сценариев фиксируйте шаги в `docs/processes/`.
